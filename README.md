# DermAI – Skin Lesion Classifier

A **Gradio** demo of a PyTorch skin lesion classifier.

## Live Demo
[https://huggingface.co/spaces/YOUR_USERNAME/dermai](https://huggingface.co/spaces/YOUR_USERNAME/dermai)

2. View top-5 predictions with confidence scores.
